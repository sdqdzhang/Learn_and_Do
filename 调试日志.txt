【环境前提】
- Python 3.12+，项目根目录已配置虚拟环境与依赖：pip install -r requirements.txt
- Node.js + npm（前端在 web/）
- 按需：Docker、Ollama、根目录 .env（可从 .env.example 复制）

--------------------------------------------------------------------------------
一、启动后端（FastAPI + WebSocket）
--------------------------------------------------------------------------------
在项目根目录（含 server/ 的目录）打开终端：

  Windows（PowerShell）:
    .\.venv\Scripts\activate
    uvicorn server.app:app --host 127.0.0.1 --port 8765

  说明：
  - 默认 CORS 允许 http://127.0.0.1:5173 与 http://localhost:5173（与 Vite 一致）
  - 健康检查：浏览器或 curl 访问 http://127.0.0.1:8765/api/health
  - WebSocket 会话：/ws/session（前端经 Vite 代理走 /ws/...）

--------------------------------------------------------------------------------
二、启动前端（Vite + React）
--------------------------------------------------------------------------------
在 web 子目录：

    cd web
    npm install
    npm run dev

  默认开发地址一般为：http://127.0.0.1:5173
  vite.config.ts 已将 /api、/ws 代理到 http://127.0.0.1:8765，故需先起后端再起前端（或先起后端再刷新页面）。

--------------------------------------------------------------------------------
三、与 main.py 的区别
--------------------------------------------------------------------------------
  python main.py --prompt "..."  为命令行单次会话入口，不是 Web 服务。
  看板联调请用上面的 uvicorn + npm run dev。
